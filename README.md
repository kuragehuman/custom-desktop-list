# custom-desktop-list
自分用の箇条書きメモ常駐アプリ

## 【基本操作】

・マウスカーソルを画面の四隅(デフォルトは左下)に寄せると表示

・入力欄にメモを入力しEnterでリストに追加

・各項目の右側の×ボタンで削除が可能

・配置位置はタスクトレイのアイコンを右クリックで選択可能(画面の四隅)

・list.json, config.jsonがAppDataに保存され、削除する場合はタスクトレイから「リスト・設定の削除」を選択

---

## 【設計メモ】

・ctrl + shift + space で開いて入力まで可能にする(現在開くが入力できない、ウインドウをアクティブにできていない？→タイトルバー付きウィンドウでは実装が難しそうなため保留)

## 【設計メモ(実装済み)】

・マウスカーソルを画面左端に動かすとリストを表示

・パネルにマウス乗っている間は閉じない　

・リストの横幅より右側にマウスカーソルがいくと隠す

・メモの追加・削除

・変更時にjsonを保存(AppDataフォルダ)

・起動時にjsonを読み込み、ない場合は作成する

・データ保存はjson(パラメータはtextのみ)

・箇条書きのリストの一番上に入力欄

・入力欄に入力してエンターで自動的に追加、入力欄クリア

・入力欄か空の場合は追加なし

・各項目の右側に削除ボタンを設置してワンクリックで削除

・常に最前面表示

・位置変更・終了はタスクトレイから

・位置をconfig.jsonに保存

・タスクトレイからリストとコンフィグの削除
