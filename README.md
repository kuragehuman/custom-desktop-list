# custom-desktop-list
自分用の箇条書きメモ

【設計メモ】

・マウスカーソルを画面左端5pxに動かすとリストを表示

・パネルにマウス乗っている間は閉じない

・最大文字数は30文字

・最大30件まで

・リストの横幅より右側にマウスカーソルがいくと隠す

・Escでも隠す

・機能はメモの追加・削除・背景色変更

・背景色変更・終了はタスクトレイから

・変更時にjsonを保存(AppDataフォルダ)

・起動時にjsonを読み込み、ない場合は作成する

・データ保存はjson(パラメータはtext, isDelete)

・箇条書きのリストの一番下に入力欄

・行ごとに若干色を変える(交互に)

・入力欄に入力してエンターで自動的に追加、入力欄クリア

・入力欄か空の場合は追加なし

・各項目の右側にゴミ箱ボタンを設置してワンクリックで削除

・ゴミ箱と、ゴミ箱を空にするボタン

・常に最前面表示
