# custom-desktop-list
自分用の箇条書きメモ

【設計メモ】

・背景色変更はタスクトレイから

・位置・背景色をconfig.jsonに保存

【設計メモ(実装済み)】

・マウスカーソルを画面左端に動かすとリストを表示

・パネルにマウス乗っている間は閉じない

・リストの横幅より右側にマウスカーソルがいくと隠す

・メモの追加・削除

・変更時にjsonを保存(AppDataフォルダ)

・起動時にjsonを読み込み、ない場合は作成する

・データ保存はjson(パラメータはtextのみ)

・箇条書きのリストの一番上に入力欄

・入力欄に入力してエンターで自動的に追加、入力欄クリア

・入力欄か空の場合は追加なし

・各項目の右側に削除ボタンを設置してワンクリックで削除

・常に最前面表示

・位置変更・終了はタスクトレイから
